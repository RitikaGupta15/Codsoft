## Description
A simple Java console game where the user guesses a randomly generated number between 1 and 100. The game provides hints (too high / too low) and tracks the number of attempts.

## Features
- Random number generation
- User input and feedback
- Limited attempts per round
- Option to play multiple rounds
- Score based on performance

## How to Run
1. Compile: `javac NumberGame.java`
2. Run: `java NumberGame`
